Privacy

VaultPass is designed to minimize data collection and avoid third-party tracking.

What we store

  • Account identifiers (email) and authentication data required to run the service.
  • Vault items you save (and optional client-encrypted fields where enabled).
  • Security/audit events for reliability and security posture.

Analytics

If usage analytics is enabled in your deployment, it is first-party and opt-in from the dashboard. It should never include vault secrets.
