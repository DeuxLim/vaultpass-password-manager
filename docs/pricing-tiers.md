# Pricing Tiers (Draft)

Status: Draft for Phase 6
Owner: VaultPass core team

This document defines a user-facing pricing model. It is intentionally lightweight and can be refined before launch.

## Goals

- Keep a generous free tier that proves value.
- Charge for collaboration and convenience features.
- Keep security features (2FA, session controls) available to all users.

## Proposed Tiers

### Free
For individuals getting started.

Includes:
- Unlimited vault items
- Password generator + health dashboard
- Import/export + encrypted backups
- 2FA + session management
- PWA support

### Plus (Individual)
For daily users who want added safety and convenience.

Adds:
- Optional breach monitoring
- Priority support

### Family / Team
For households and small teams.

Adds:
- Shared vaults (multiple shared vaults)
- Emergency access
- Invite notifications (email / push where enabled)

## Notes

- Do not paywall core security (2FA, session revoke).
- Billing should never block access to existing data. If a plan lapses, move the account into a “read-only” or “limited collaboration” mode rather than hard-locking the vault.
